The Brief

Heineken was launching a new, centralised B2B eCommerce platform that was consolidated, yet customisable for each individual country/region. My role in this project was to implement new features and improve the current UX/UI where feasible for relevant Opcos in the APAC region.

Platform
Web & App (App focus).

Deliveries
Interactive Prototype.

CHALLENGES

Requirements of some Opcos can be challenging to customise so that the features would meet the Opcos' needs while being well-aligned with the central platform. Thus, any design decision was a result of many discussions amongst multidisciplinary teams.

On-boarding Flow

Problem(s):
• Unfriendly Login/Registration processes, too much information on a single screen.
• The UI was not optimised for mobile (unsuitable font-size, unnecessary use of overlays, distracting background, and so on).
• Improvement(s):
• Reviewed and trimmed down unnecessary information, broke down the process into smaller steps: 1 field per screen.
• Limited options: multiple inputs accepted in the same field (i.e. email, username or phone number for login ID).
• Auto validation (validate as soon as the user has done typing - no additional click needed to proceed).
• Simple, lean UI.

Multi-distributor Order Flow

Problem(s):
• Current system only allowed purchasing from 1 distributor per time. As an eCommerce platform, this wouldn't provide optimal experience for users.
• Distributor could only be selected at the beginning of app/web launch, no direct or easy path to re-select.
• Compromised current cart logic when changing distributor while there were already items from another distributor in cart.
Improvement(s):
• Revised the whole ordering experience
• Revamped UI of: Product Category/Listing - Cart - Order Summary - Checkout - Confirmation.
• Incorporated distributor selection on Category Browsing for easy switching.
• Catered Cart, Order Summary, Checkout, Confirmation for multiple distributor display.
• Options to review & select/deselect items on Cart.

Order Details & Management Flow

Problem(s):
• Order Details and Management needed to be customised to suit individual Opco's mechanism.
Improvement(s):
• Analysed requests vs. supported UX/UI and updated/created designs accordingly.
