The Brief

Identified UX/UI problems and user pain points, redesigned user flows, and created interfaces for new features for the centralised B2B platform, customising for individual countries/OpCos within the APAC region and optimising webapp and app versions

Project Type
B2B eCommerce, webapp & mobile app

Deliverables
Interactive prototypes & mockups

CHALLENGES

Balancing stakeholder demands and system consistency, capacity, resources, and development efforts. Potential complications & conflicts with existing builds. Support & optimisation for older, non-tech-savvy audience (main users). Complex, uncommon interfaces vs. firt-time users

On-boarding Flow

Task(s):
Optimise Login and Registration process; redesign onboarding flows (Login, Register, Forgot Password), applying UX best practice; refine and enhance UI using mobile-first approach

Approach(es):
• UX research based on trend and market reports
• Constructed detailed flow diagram for all user flows
• Identified users' pain points of traditional onboarding
• Simplified the experience, reduced taken steps, limited the amount of information displayed, eliminated distractions
• Phone number and OTP login/register integration to streamline the login and onboarding processes
• Minimise input fields per screen, supporting flexible input types in 1 single field
• Simple, lean UI, accommodating bigger text display

Ordering Experience

Task(s):
Redefine and amend complex order flows for individual user types, ensuring consistency and integrity between the flows; integrated new functions (e.g. different order types, specified orders, orders with equity, different checkout methods); tailor/add functionalities for individual OpCos

Approach(es):
• Investigated existing conflicts between user flows and assessed potential ones
• Enhanced logic between interactions
• Redesigned user flows
• Created details flows for micro interactions
• Identified and resolved discrepancies and incoherence between flows
• Enhanced checkout process & information hierachy
• Revamped UI for clearer/more transparent displays
