Radio Rentals

Radio Rentals is one of the largest appliance and electronics lenders in Australia. I participated in the UX/UI enhancement phase to propose UX solutions and create Customer Profile and Online Rental Application forms for optimal user experience.

Project type
Website design.

My Role
UX/UI enhancements.

THE CHALLENGE

As a lender platform, it is very important that customers have a fluid experience when applying for rentals. I worked with the user research team to construct a seamless application process and an organised user profile that met customer needs. Every micro interaction mattered.

The Application

Each step of the application process was designed base on insights from user research and client's requirements. The process required quite many inputs and verifications which could be a hassle for customers if not presented well. We agreed to break the process down to step by step and arranged the inputs by common order and their importance (Personal -> Residental -> Income & Expenses -> Reference).

The Profile

Customer Profile was constructed under similar layout to the Online Rental Application for UI consistencies, and to minimise development efforts. Customer Profile also contained all rental/order/application information and details which could be very content-heavy. Therefore, we organised relevant details in expandable panels and implemented paging for data loading efficiency.
