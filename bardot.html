The Brief

Optimise/review the whole site's UX/UI and deliver a new website to enhance user shopping experience and sale revenues.

Project type
Website re-design.

My Role
UX/UI design.

CHALLENGES

Many micro interactions, out-of-scope functionalities, mobile optimisation for heavy contents.

SEARCH UI Enhancement

Search is an important function for most platforms, even more so for eCommerce. The first approach in the redesign process was to reconstruct the header, with a focus on the Search function. The new search UI is a lot more prominent as opposed to the old generic one, with 'pre-suggested' trending items serving as a visual cue to draw user attention to 'hot' products.

SEARCH SUGGESTION UX

The search suggestion was also studied carefully by our UX team, we analysed common keywords and proposed that the suggestions should go beyond matching keywords to:
1. Always show suggestions (even if the keyword goes wild)
2. Pushed sale items to be prioritised
3. Categorised suggestions base on keyword(s)

Header Revamp

Decluttered old header by removing unnecessary items, made mandatory ones accessible (base on heatmaps from user research).

DROPDOWN MENU UI

Enhanced dropdown menu visual and purposely added relevant promotional CTA banners to make use of the header's real estate.

LOCATE IN-STORE Feature

An out-of-scope function to show which stores the item is currently available at. Designed as a popup for a quick check-and-close action before the users continue with the browsing.

Store finder Design

Store finder design with store-related promotion panels.

PRODUCT BADGE Design

Unique product badges for applicable product tiles.

Secondary Pages

All secondary pages: Order Details/Management, Account Management, Settings, etc.
